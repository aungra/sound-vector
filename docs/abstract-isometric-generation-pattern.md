# Abstract Isometric One-Ink Pattern

Reference analysis for the T-shirt dummy designs.

## Visual Grammar

- Use one ink only: black ink on white/unprinted shirt.
- Build objects from white top faces, black side faces, and black cast shadows.
- Prefer soft rounded forms: cylinders, rings, capsules, rounded slabs, arches.
- Compose each design as a small abstract object system on a platform, not as a literal scene.
- Use isometric cues with offset top and side faces, but avoid heavy wireframe grids.
- Add small floating blocks, dots, short bars, and shadows to create rhythm.
- No visible English/Japanese text in the artwork.

## Construction Rules

- Main form: one large platform or slab, usually elliptical or rounded rectangular.
- Secondary form: 2-4 stacked blocks, cylinders, or rings.
- Accent form: small floating cubes/capsules/dots.
- Shadow: black offset shape under the main form.
- Detail: thin black strokes on white faces, plus a few black side fills.
- Keep enough white interior space so the design does not become a black mass.

## Print Rules

- All visible ink is `#111`.
- White is substrate/negative space only.
- No gradients, gray, transparency, or image textures.
- Use line widths around 12-18px for silkscreen stability.
